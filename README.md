virtualjaguar-libretro
======================

Port of Virtual Jaguar to Libretro

Upstream: `git clone http://shamusworld.gotdns.org/git/virtualjaguar`

Unofficial GitHub mirror: https://github.com/mirror/virtualjaguar

Needs battery saves

JOY1/JOY2 :

L3  BUTTON 4
R3  BUTTON 3
L2  BUTTON 2 
R2  BUTTON 1
L   BUTTON S
R   BUTTON D
SEL BUTTON OPTION
STR BUTTON PAUSE
A   BUTTON A
B   BUTTON B
X   BUTTON 0
Y   BUTTON C
U/D/L/R Joy dir
